# 🍆 Eggplant Delights - ナス料理ギャラリー

モダンでスタイリッシュなナス料理を紹介する、インタラクティブなシングルページWebアプリケーションです。

![Eggplant Delights](./001-Generated%20Image%20November%2014,%202025%20-%209_40PM.png)

## ✨ 特徴

- 📸 **10種類の絶品ナス料理** - 美しい画像と詳細なレシピ
- 🎨 **モダンなデザイン** - グラデーション、アニメーション、シャドウ効果
- 🌙 **ダークモード対応** - ワンクリックでテーマ切り替え
- 🔍 **カテゴリーフィルター** - グリル、オーブン焼き、揚げ物、煮込みで絞り込み
- 📱 **完全レスポンシブ** - PC、タブレット、スマホに対応
- ⚡ **スムーズなアニメーション** - スクロール連動のフェードイン効果
- 🖼️ **インタラクティブなモーダル** - クリックで詳細レシピを表示

## 🍽️ 収録レシピ

1. **とろけるガーリックバターグリル** - ガーリックバターの香りが絶品
2. **ナスのラザニア** - グルテンフリーのイタリアン
3. **ナスの照り焼きステーキ** - 和風の定番人気メニュー
4. **ナスとモッツァレラのカプレーゼ** - 色鮮やかな前菜
5. **ナスのマイルドキーマカレー** - 辛さ控えめで誰でも楽しめる
6. **ナスのカポナータ** - 地中海風の甘酸っぱい煮込み
7. **ナスのハニーチーズロースト** - 甘じょっぱい創作料理
8. **ナスのマイルド麻婆** - 子供にも優しい中華
9. **ナス入りミートボールサブ** - ボリューム満点のサンド
10. **ナスのクリスピーフライ** - カリカリのヘルシースナック

## 🚀 使い方

### ローカルで実行

1. リポジトリをクローン:
```bash
git clone https://github.com/あなたのユーザー名/eggplant-delights.git
cd eggplant-delights
```

2. ローカルサーバーを起動:
```bash
# Pythonを使用する場合
python -m http.server 8000

# Node.jsを使用する場合
npx http-server

# VS Code Live Serverを使用する場合
# index.htmlを右クリック → "Open with Live Server"
```

3. ブラウザで開く:
```
http://localhost:8000
```

### GitHub Pagesでデプロイ

1. GitHubリポジトリの Settings → Pages
2. Source を `main` ブランチに設定
3. 公開URLにアクセス
